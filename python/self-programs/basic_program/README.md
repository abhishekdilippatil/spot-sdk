# basic-program

This is a program for SPOT. He performs two basic functions such as standing and taking an image from left fisheye camera.
Also you do not need an external e-stop service for this, as it automatically releases e-stop after powering on the robot and engages e-stop when the program has performed all it's tasks.

## Setup Dependencies

See the requirements.txt file for a list of python dependencies which can be installed with pip using the command:

```
python3 -m pip install -r requirements.txt

```
## Run the programme

To run the programme:

```
python3 basic_program.py ROBOT_IP
